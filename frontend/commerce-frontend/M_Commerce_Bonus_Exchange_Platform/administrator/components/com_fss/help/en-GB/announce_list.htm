Available Fields:
{class} : Ensure that this tag is on the main div of the HTML within its class attribute
{editpanel} : Displays the edit announcement buttons. Must be somewhere in the HTML to allow the editing the announcements on the frontend. If the user is not allowed to edit this announcement then its not displayed.
{date} : Date of publishing
{link} : Link to main announcement
{title} : Title of announcement
{subtitle} : Module subtitle
{author} : Authors Name
{author_username} : Authors username
{body} : Main body (before page break)
{needsreadmore} : Set to 1 if the announcement has more text or comments
{fulltext} : Remainder of main body
{readmore} : Read more text (depends on what comments are available)
{commentcount} : Count of current comments on the announcement
