The following can be used in this template:

  • {moderated} - Set if the comment requires moderation
  • {email} - EMail address
  • {website} - Website
  • {linkmod} - Link to the moderation of the comment
  • {linkart} - Link to the article or product the comment is for
  • {article} - The title of the article the comment is for
  • {name} - The name of the comment poster
  • {article_type} - The type of comment being submitted

You can also do {product_start}Product: {product}{product_end} in the template. This will only be output if the field {product} is not empty. This is usefull for sometimes hiding fields. (You can use this to hide custom fields that have no value etc).
